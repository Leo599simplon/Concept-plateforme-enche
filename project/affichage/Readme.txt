Cette partie se concentre sur l'affichage. Elle ne fait presque aucun calcul et se contente de récupérer 
des variables pour savoir ce qu'elle doit afficher. On y trouve essentiellement du code HTML mais aussi 
quelques boucles et conditions PHP très simples, pour afficher par exemple une liste de messages.
C'est le composant graphique de l’interface qui permet de présenter les données du modèle à l’utilisateur.
